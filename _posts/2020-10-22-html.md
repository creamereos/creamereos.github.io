### 시간 표 만들기

- rowspan="n" n만큼 세로 cell 차지

- colspan="n" n만큼 가로 cell 차지

- scope = "col" / scope = row (th에만 사용 가능. 브라우저에 전달)

- thead, tbody, tfoot 태그 (브라우저에 전달)

<table>

# HTML media

### img
~~~
<img src = "" alt=""/>
~~~

### audio
~~~
<audio src="" controls></audio>

[attribute]
src : 경로 (필수)
controls : 컨트롤 설정(소리,재생)
autoplay : 자동재생
loop : 반복 재생
~~~
<audio src="" controls></audio>

### video
~~~
<video src="" controls></video>
~~~
